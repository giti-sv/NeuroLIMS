# NeuroLIMS

NeuroLIMS is an interactive neuroscience learning and simulation platform designed to help users explore brain anatomy, understand neurotransmitters, and experiment with basic neural stimulation concepts in an engaging web-based environment.

## Project Overview
This project is being developed as part of a Software Engineering course. The goal is to create an educational and interactive platform that combines neuroscience learning with simple simulation tools.

## Features
- Interactive brain anatomy exploration
- Information on major neurotransmitters
- Neural stimulation experiment interface
- Save and review experiment results
- Educational dashboard for neuroscience concepts
